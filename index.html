Привет!

Это приложение позволяет отслеживать прогресс твоего ребёнка и видеть список развивающих активностей, актуальных для его возраста. Все персональные данные сейчас хранятся в твоём браузере, что обеспечивает полную приватность. Позже функционал расширится, и потребуется регистрация, но сейчас всё работает без регистрации прямо на твоём устройстве.

Чтобы весь функционал работал в полном объёме, нужно:

  1. создать профиль ребёнка (кнопка наверху) и указать его дату рождения;
  2. установить галочки напротив всех навыков, которые ребёнок к настоящему моменту уже освоил.

Для каждого навыка в приложении сохранена подробная информация: суть навыка, диагностические признаки, возможные проблемы при освоении, доступные родителю активности для помощи в развитии и т.д. Чтобы увидеть информацию по навыку, нужно кликнуть по его названию в таблице.

Чтобы развернуть группу навыков (крупная моторика, мелкая моторика и т.п.), нужно нажать на название группы.
